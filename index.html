Framework web Python para desenvolvimento rápido e seguro de aplicativos.

Sistema de gerenciamento de banco de dados relacional amplamente utilizado para armazenar e gerenciar dados.

NodeJS

Plataforma de desenvolvimento de código aberto para executar JavaScript no lado do servidor.

Um framework web para Node.js que simplifica o desenvolvimento de aplicativos web e APIs. Oferece uma variedade de recursos para roteamento, middleware e integração de banco de dados. É uma escolha popular para construir aplicativos web escaláveis e eficientes

PostgreSQL

Python

Html5
